🎓 Faculty Management System

ICT2132 – Object Oriented Programming Practicum Mini Project
University of Ruhuna – Faculty of Technology


📌 Project Overview
This project is developed for the course ICT2132 – Object Oriented Programming Practicum at the University of Ruhuna.
The system is a Java + MySQL based Faculty Management System designed to manage:

01.User Profiles (Admin, Lecturer, Student, Technical Officer)
02.Course Details
03.Undergraduate Marks
04.Undergraduate Attendance
05.Notices
06.Timetables
07.Medical Records
08.GPA & Eligibility Calculations

This system follows Object Oriented Programming (OOP) concepts and includes GUI and database connectivity.

🛠️ Technologies Used

-Java (OOP Concepts)
-MySQL Database
-JDBC (Database Connectivity)
-Java Swing / JavaFX (GUI)
-Git & GitHub (Version Control)

👥 User Roles & Functionalities

🔐 Admin
-Create & manage user profiles
-Create & manage courses
-Manage notices
-Manage timetables

👨‍🏫 Lecturer
-Update profile (except username & password)
-Add course materials
-Upload marks (out of 100)
-View undergraduate details
-Check eligibility (Attendance + CA)
-View GPA, Grades
-View attendance & medical records
-View notices

🛠️ Technical Officer
-Update profile (except username & password)
-Manage attendance
-Manage medical records
-View notices
-View department timetables

🎓 Undergraduate
-Update contact details & profile picture
-View attendance
-View medical status
-View course details
-View grades & GPA
-View timetables
-View notices
